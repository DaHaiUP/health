我们需要通知权限用于：

  • 设置定时提醒闹钟，确保按时提醒

  • 管理多个提醒时间点，提供灵活的提醒服务

4. 主机权限 (Host Permissions - Content Scripts)

我们需要主机权限以便：

  • 实现跨页面的状态同步，保持一致的用户体验

The alarm permission is used for:

We need host permissions to:

  • Implement cross-page state synchronization to maintain a consistent user experience
